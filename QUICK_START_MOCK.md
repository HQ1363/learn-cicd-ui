# 🚀 快速开始 - Mock 模式

## 一分钟上手

### 1️⃣ 启动开发服务器

```bash
npm install
npm run dev
```

或者使用 yarn：

```bash
yarn install
yarn dev
```

### 2️⃣ 打开浏览器

访问：`http://localhost:8080` (或配置的端口)

### 3️⃣ 开始使用

✅ **无需登录** - 页面自动进入应用首页  
✅ **无需后端** - 所有数据都是测试数据  
✅ **完整功能** - 支持流水线的增删改查  

## 快速演示

### 创建流水线

1. 点击右上角"新建流水线"按钮
2. 填写流水线信息：
   - 名称：我的测试流水线
   - 描述：这是一个测试流水线
   - 选择应用和环境
3. 点击"确定"
4. ✅ 流水线创建成功，立即显示在列表中

### 编辑流水线

1. 找到任意流水线
2. 点击右侧"操作"菜单
3. 选择"编辑"
4. 修改信息后保存
5. ✅ 更改立即生效

### 删除流水线

1. 找到要删除的流水线
2. 点击右侧"操作"菜单
3. 选择"删除"
4. 确认删除
5. ✅ 流水线从列表中消失

### 收藏流水线

1. 找到喜欢的流水线
2. 点击"收藏"按钮
3. ✅ 在"我收藏的"标签页查看

### 克隆流水线

1. 找到要克隆的流水线
2. 点击"克隆"按钮
3. 输入新名称
4. ✅ 创建一个完全相同的副本

## 重要提示

### ⚠️ 数据不会保存

Mock 数据存储在浏览器内存中，刷新页面后会重置：

- ✅ 创建的流水线 → 刷新后消失
- ✅ 修改的内容 → 刷新后恢复
- ✅ 删除的数据 → 刷新后回来

这是正常现象，因为没有连接真实的数据库。

### 🔄 如何重置数据

如果数据被改乱了，想要恢复初始状态：

**方法 1：刷新页面**
```
按 F5 或点击浏览器刷新按钮
```

**方法 2：清除缓存**
```
Ctrl + Shift + Delete (Windows)
Cmd + Shift + Delete (Mac)
```

### 🐛 遇到问题？

如果遇到问题，先尝试：

1. **刷新页面** - 解决 90% 的问题
2. **查看控制台** - 看看有没有错误日志
3. **清除缓存** - 清除浏览器缓存后重试
4. **重启服务** - 停止开发服务器，重新启动

### 🎯 常见问题

**Q: 为什么看不到登录页面？**  
A: 开发环境下已经绕过登录，直接进入应用。

**Q: 我创建的流水线刷新后消失了？**  
A: Mock 数据存在内存中，刷新会重置。这是正常的。

**Q: 如何连接真实后端？**  
A: 在控制台执行 `window.__restoreMock()` 可以临时恢复真实请求。

**Q: 为什么某些功能不能用？**  
A: 部分功能需要后端支持（如流水线执行、实时日志等）。
